Why Use CSS?

Cascading Style Sheets tells the browser how to diplay its content that we have writen.

We can use CSS selectors to target HTML elements and apply a variety of attributes on elements to change how its displayed.

CSS Anatomy

A CSS rule is made up of a selector, its declaration

A declaration is made up of a property and a property value

Attribute Selectors

A special selector that selects an element based on its attribute & attribute value.

[attribute="value"]{
background-color: blue;
}
